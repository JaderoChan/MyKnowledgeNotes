# [C/C++] 程序运行内存分区详解 *

**[AI 回答，待提炼]**

程序运行时，操作系统会给进程划分5大核心内存区，从上到下地址排布：
栈区 → 堆区 → 全局/静态区 → 常量区 → 代码区

顺带把每个区存放内容、生命周期、读写权限、特点一次性讲清。

1. 代码区（Text / 代码段）

    **存放内容**

    - 所有二进制机器指令
    - 函数体代码、if/for/循环、调用逻辑

    **特点**

    1. 只读权限，运行中不能修改
    2. 多个进程可共享同一份代码，节省内存
    3. 程序结束才释放

    **存放示例**

    ```c
    void func(){}    // 代码放这里
    int main(){...}  // 代码放这里
    ```

2. 常量区（只读数据段 RO Data）

    **存放内容**

    - 字符串常量
    - const 修饰的全局常量
    - 字面常量

    **特点**

    1. 只读，强行修改直接崩溃报错
    2. 程序结束释放
    3. 全局共享

    **示例**

    ```c
    char *str = "hello";  // "hello" 存在常量区
    const int g_num = 10; // 全局const常量在这里
    ```

    ⚠️ 不能  str[0]='a' ，常量区禁止写

3. 全局/静态区（Data + BSS）

    整块统称静态存储区，细分两段

    1. 初始化全局数据段（Data 段）

        存放：已初始化的全局变量、static 静态变量

        - 普通全局变量
        - static 局部变量
        - static 全局变量

    2. 未初始化全局段（BSS 段）

        存放：未初始化全局变量
        程序启动统一默认赋值 0

    **共同特点**

    1. 程序整个运行周期存活
    2. 程序结束才释放
    3. 可读可写
    4. 所有函数、对象共享

    **存放示例**

    ```c
    int g_a = 10;        // Data 段
    int g_b;             // BSS 段
    void test(){
        static int s = 5;// 静态区 Data 段
    }
    ```

4. 栈区（Stack）

    **存放内容**

    1. 局部普通变量
    2. 函数形参、返回地址
    3. 函数调用现场、临时变量

    **特点**

    1. 自动分配、自动释放
    2. 函数调用时开辟，执行完立刻销毁
    3. 空间小、读写速度极快
    4. 地址从上往下增长
    5. 超出大小会栈溢出

    **示例**

    ```c
    void fun(int x){  // x 栈区
        int a = 1;    // a 栈区
        double b = 2; // b 栈区
    }// 函数结束，a b x 全部销毁
    ```

5. 堆区（Heap）

    **存放内容**

    程序员手动申请的动态内存

    - C：malloc / calloc / realloc
    - C++：new / new[]

    **特点**

    1. 手动申请、手动释放，不 free/delete 会内存泄漏
    2. 生命周期由程序员决定
    3. 空间极大，适合存大数据、数组、对象
    4. 地址从下往上增长
    5. 没有自动回收

    **示例**

    ```c
    int *p = (int*)malloc(4); // 堆内存
    free(p); // 必须手动释放
    ```

    ```cpp
    int *arr = new int[100];
    delete[] arr;
    ```

    额外补充：寄存器内存（CPU 高速存储）

    不属于进程内存，但程序运算必用

    - 存放：频繁运算变量、中间结果
    - 速度最快，容量极小
    - volatile 就是禁止把变量常驻寄存器

## 一张表汇总所有分区

| 分区 | 存放内容 | 生命周期 | 读写 | 管理方式 |
| ---- | -------- | ------- | --- | ------- |
| 代码区 | 程序指令、函数代码 | 程序全程 | 只读 | 系统自动 |
| 常量区 | 字符串常量、全局 const | 程序全程 | 只读 | 系统自动 |
| 全局静态区 | 全局变量、static 变量 | 程序全程 | 读写 | 系统自动 |
| 栈区 | 局部变量、形参 | 函数调用期间 | 读写 | 自动创建销毁 |
| 堆区 | malloc/new 动态内存 | 手动控制 | 读写 | 手动申请释放 |
