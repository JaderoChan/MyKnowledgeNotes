# [Algorithm] SHA-1 计算

SHA‑1 是密码学哈希函数，输入任意长度数据，输出固定 160 bit 摘要。

## 步骤

1. 填充数据
2. 初始化缓冲区
3. 分块处理
4. 输出结果

## 填充数据

根据标准 **FIPS 180-1**，需要先填充数据使输入数据的长度为 512 bit 的整数倍。其具体步骤如下：

1. 追加一个 `1` bit。
2. 追加若干 `0` bit。
3. 追加原数据长度（64 位大端序）。

因为原数据长度是固定追加 64 位，所以对于步骤 1 和步骤 2 而言，其需要填充至的数据长度 `length` 应满足 `length % 512 == 448`。

## 初始化缓冲区

SHA-1 使用 5 个固定的初始值开始计算：

```plaintext
H0 = 0x67452301
H1 = 0xEFCDAB89
H2 = 0x98BADCFE
H3 = 0x10325476
H4 = 0xC3D2E1F0
```

## 分块处理（每 512 bit 一块）

对每块执行 80 轮循环运算：

1. 扩展成 80 个 32‑bit 字
2. 4 轮不同逻辑函数（f₀~f₇₉）
3. 循环移位、与或非、模 2³² 加法
4. 每轮更新 H0~H4

## 输出结果

把 H0 H1 H2 H3 H4 拼接 → 160 bit 哈希值

（可选）→ 转成十六进制 → 40 位字符
